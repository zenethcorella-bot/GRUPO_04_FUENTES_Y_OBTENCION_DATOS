# GRUPO_04_FUENTES_Y_OBTENCION_DATOS


# Obtención y Preparación de Datos Financieros a partir de la API Yahoo Finance.

## Descripción del proyecto
Este repositorio contiene el desarrollo del trabajo correspondiente a la Asignatura 5 del máster.  
El objetivo del proyecto es obtener, limpiar y analizar datos financieros reales utilizando una API, como base para el desarrollo del Trabajo Fin de Máster (TFM).

## Fuente de datos
Los datos han sido obtenidos mediante la API de Yahoo Finance utilizando Python y la librería `yfinance`.


## Contenido del repositorio
- Notebook `.ipynb` con:
  - Extracción de datos desde la API  
  - Limpieza y preparación de datos  
  - Análisis exploratorio (EDA)  
  - Visualización de resultados  

## Requisitos técnicos
Para ejecutar el notebook se requieren las siguientes librerías:
- yfinance  
- pandas  
- matplotlib  
- seaborn  

## Reproducibilidad
El notebook puede ejecutarse de forma secuencial desde la primera celda hasta la última.
